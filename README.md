## Features

* Cursor-reactive ASCII typography
* Particle-based text animations
* Generative motion systems
* Interactive hover and distortion effects
* Custom ASCII rendering pipeline
* Responsive layouts and full-screen experiences

## Stack

* React
* TypeScript
* Three.js
* React Three Fiber
* GLSL Shaders
* Tailwind CSS

## Overview

TYPE 001 investigates how typography can function as both content and interface. Letters dissolve into particles, respond to movement, and transform through a collection of interactive visual experiments.

Each piece explores a different relationship between text, motion, and user input, using ASCII characters as the primary visual material.

## Development

```bash
npm install
npm run dev
```
